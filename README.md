# DigiTools 🚀

DigiTools is a simple ReactJS project for showing premium digital tools in a clean and responsive UI.  
It has a product section, cart section, pricing cards, and a modern landing page design.

## Live Site
[https://digitools-web-araf.netlify.app/](https://digitools-web-araf.netlify.app/)

## 🧰 Tech Stack

- **React**
- **Vite**
- **Tailwind CSS**
- **DaisyUI**
- **React Icons**
- **React Toastify**

## Features
- Responsive landing page
- Product cards with pricing
- Add to cart option
- Cart item count in navbar
- Prevents duplicate items in cart
- Remove item from cart
- Auto total price calculation
- Checkout button with toast message
- Pricing section
- Simple and clean UI

## Shopping Cart
- Click **Buy Now** to add a product
- Same product cannot be added twice
- Cart count updates instantly
- You can remove products from the cart
- Total price updates automatically
- Clicking checkout clears the cart and shows a success message

## Run Locally

```bash
git clone https://github.com/mi-araf/B13-A6-DigiTools-web.git
cd B13-A6-DigiTools-web
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```
